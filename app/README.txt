Welcome to the ArcGIS Maps SDK for Java!

v200.3.0

----------------------------
Overview
----------------------------

The ArcGIS Maps SDK for Java provides you with the API libraries required to build apps for ArcGIS Java.

----------------------------
SDK Contents
----------------------------

The ArcGIS Maps SDK for Java contains everything you need to develop ArcGIS Java applications.
The contents of the SDK are provided below:

 + jniLibs > All the platform native libraries (Windows, MacOS, Linux)
 + legal > All the SDK license files.
 + libs > API jar libraries for arcgis-java and third party dependency jar libraries.
 + resources > Resources to work with the SDK.

----------------------------
Get started
----------------------------

1. Copy the jniLibs and resources folders from the SDK into the root directory of your project.
2. Add the jars in the libs folder to your classpath.
